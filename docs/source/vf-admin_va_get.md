## vf-admin va get

Retrieve a vaccine availability with a specified id

```
vf-admin va get [flags]
```

### Examples

```
# Get the vaccine availability with id 014cc133-484f-4320-be3b-444e758b64a7
$ vf-admin va get 014cc133-484f-4320-be3b-444e758b64a7

```
